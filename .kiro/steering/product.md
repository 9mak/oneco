# Product Overview

Kiro AI-DLC (AI Development Life Cycle) - スペック駆動開発フレームワーク

## Core Capabilities

- **段階的仕様策定**: 要件 → 設計 → タスク → 実装の3段階承認フロー
- **プロジェクト記憶**: `.kiro/steering/` による永続的なプロジェクト知識管理
- **仕様管理**: `.kiro/specs/` による個別機能の形式化された開発プロセス
- **品質検証**: 設計レビュー、ギャップ分析、実装検証の組み込みツール

## Target Use Cases

- AI支援による構造化された開発プロセス
- 大規模機能の段階的実装計画
- プロジェクト全体の一貫性維持
- 要件から実装までの追跡可能性確保

## Value Proposition

Claude Code との統合により、AI エージェントが段階的かつ構造化された方法でソフトウェア開発を支援。人間のレビューポイントを明確化し、スコープクリープを防止しながら、プロジェクト全体の文脈を維持。
