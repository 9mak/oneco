# Japan TopoJSON Attribution

`japan.topojson` is sourced from:

- Repository: https://github.com/dataofjapan/land
- File: `japan.topojson`
- Properties: `nam` (English), `nam_ja` (Japanese), `id` (prefecture ID)

geometry derives from public Japanese government data (国土数値情報).
